# 🤖 AI-Powered Mock Interview Trainer

An interactive AI-based system that helps users prepare for job interviews through real-time practice, intelligent feedback, and performance analysis.

---

## 📌 Overview

The **AI-Powered Mock Interview Trainer** simulates real interview environments by generating questions based on user preferences such as job role, skills, or domain. Users can respond via text or voice, and the system evaluates their answers using AI to provide constructive feedback.

This project aims to improve interview readiness, communication skills, and confidence through continuous practice and personalized suggestions.

---

## 🚀 Features

- 🎤 Interactive Interview Simulation (Text / Voice-based)  
- 🤖 AI-Generated Questions based on role & skills  
- 📝 Intelligent Answer Evaluation  
- 📊 Performance Analysis (Confidence, Clarity, Relevance)  
- 🔁 Multiple Interview Rounds (HR, Technical, Behavioral)  
- 💡 Personalized Feedback & Suggestions  
- ⚡ Real-time AI Response Processing

## ⚙️ How It Works

1. User selects job role or inputs preferences  
2. System generates interview questions using AI  
3. User answers via text or voice  
4. AI analyzes responses based on:
   - Relevance  
   - Confidence  
   - Communication quality  
5. Feedback and improvement suggestions are provided  
6. Final performance summary is generated

## 🔧 Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/mock-interview-trainer.git
cd mock-interview-trainer
```

### 2️⃣ Create Virtual Environment
```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 4️⃣ Set API Key
Create a `.env` file and add:
```
GEMINI_API_KEY=your_api_key_here
```

### 5️⃣ Run the Application
```bash
python app.py
```
